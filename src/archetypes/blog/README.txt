Blog
----

This archetype is based on the "Blog" layout provided by the 
purecss framework at http://purecss.io/layouts/ .

Structure
---------

The blog is defined as a multilanguage one. If you don't need it, you can
modify the configuration:

 - set your locale
 - in content/posts/* remove all the posts
 - modify the base template in common/base-page.peb
